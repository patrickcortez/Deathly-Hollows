# Deathly Hollow

A text adventure game where A man tries to escape from a cult.

## Set up

1. You must have `gcc` to compile the game.

2. clone the repository.

3. `cd` into the repository.

4. Or just simply `chmod +x check.sh` then run `./check.sh`

## Compile:

```powershell
# execute the script to compile the game
./check.bat
```

## Run

```bash

cd bin && ./Desp

# or

./bin/Desp

```
